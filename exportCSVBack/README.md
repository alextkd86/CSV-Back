# README #
BACK-END:
	- NODEJS.
	- EXPRESSJS.
	- MONGODB.

Procedimiento para arrancar este proyecto en local:
	1. Tener instalado NodeJS.
	2. Tener instalado ExpressJS.
	3. Tener instalado MondoDB.
	4. Tener instalado Gulp.
		- npm install gulp-cli -g
		- npm install gulp -D
	5. Arrancarlo:
		--> gulp
